# CodeAlpha_EDA_RetailSales

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset using Python.

## Objectives
- Analyze sales performance
- Identify profitable categories and products
- Study regional sales trends
- Generate business insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Findings
- Technology generated the highest sales and profit.
- Office Supplies showed strong profitability.
- Several technology products were identified as top profit contributors.
- Regional analysis highlighted high-performing markets.
